4. Changing the datatype. Here is the Query.

CREATE TABLE data_new (
    InvoiceNo TEXT,
    StockCode TEXT,
    Description TEXT,
    Quantity TEXT,
    InvoiceDate TEXT,
    UnitPrice TEXT,
    CustomerID TEXT,
    Country TEXT
);

INSERT INTO data_new
SELECT
    InvoiceNo,
    StockCode,
    Description,
    Quantity,
    InvoiceDate,
    UnitPrice,
    CustomerID,
    Country
FROM data;


verifying

SELECT COUNT(*) FROM data;
SELECT COUNT(*) FROM data_new;

drop the old table

5. generating the dump file

sqlite3 mydb.db .dump > dump.sql
